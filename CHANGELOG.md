## [v0.1.0] - 2026-03-14

### Highlights

- Pure-Go font glyph coverage checker for Pandoc/LaTeX document validation
- Drop-in replacement for fc-list font discovery with structured output

### Added

- `fontscan check` command to verify font supports all characters in a document ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `fontscan list` command to list and filter available fonts with `--filter`, `--family-only`, and `--limit` flags ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `fontscan info` command to display font information with optional `--coverage` for Unicode block analysis ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `fontscan recommend` command to find fonts supporting all characters in a document ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `fontscan pandoc` command to parse YAML frontmatter and validate font settings ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `font` package for loading TTF, OTF, and TTC fonts with glyph coverage analysis ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `scanner` package for cross-platform font discovery (macOS, Linux, Windows) ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `checker` package for document-to-font compatibility validation ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `pandoc` package for Pandoc frontmatter parsing with default font database ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- `format` package with TOON (default), JSON, and JSON-compact output formats ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- Unicode character names in missing character reports (e.g., "U+2192 RIGHTWARDS ARROW") ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- 36 common Unicode block coverage analysis ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- Flexible font name matching (case-insensitive, space-insensitive) ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
- Exit code 1 when missing characters found for CI/CD integration ([`2c9693a`](https://github.com/grokify/fontscan/commit/2c9693a))
